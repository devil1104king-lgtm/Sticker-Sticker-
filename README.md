# Sticker Chat Game Bot

A Telegram bot that allows users to play a sticker game: they send a sticker, and the bot replies with a random sticker from the selected category.

## Features

- Force join: Users must join a required channel before using the bot.
- Welcome menu with category selection.
- Sticker game: reply with random stickers from chosen category.
- Private logging group: logs all sticker activity.
- Admin panel with owner-only commands:
  - Add/remove stickers
  - Create/delete categories
  - Broadcast messages
  - View statistics
  - Ban/unban users
  - Change channel / log group
  - Export and backup database
